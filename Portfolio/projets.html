🎉 Just for fun

Just for fun est un dépôt GitHub qui regroupe plusieurs mini-sites web créés pendant mon temps libre. Ces projets sont réalisés en HTML, CSS et JavaScript et explorent différentes idées autour de :

  • 🎨 Animations et effets visuels (transitions, transformations, micro-interactions)
  • 🕹️ Interactivité (petits jeux, interactions utilisateur, logique front-end)
  • 💡 Expérimentations créatives (design, UI intuitive, concepts originaux)

Ce projet est né de l’envie d’apprendre, de progresser et de m’amuser en codant. Chaque mini-site est indépendant et peut être ouvert directement dans un navigateur pour une découverte rapide.
