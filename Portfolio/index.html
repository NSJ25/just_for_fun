Présentation

Bonjour,

Je suis The Ghostwriter 25, étudiant en informatique basé en Belgique.

Passionné par l’informatique dans son ensemble, je m’intéresse à ses multiples domaines : développement web, programmation, cybersécurité, intelligence artificielle, systèmes d’exploitation… Tout m’inspire et m’intrigue.

Je me forme activement en HTML, CSS, JavaScript et Python, et je réalise des projets personnels pour mettre en pratique ce que j’apprends.

En dehors du code, je suis aussi passionné par les citations, ces phrases simples qui portent parfois des vérités puissantes.

Ce portfolio me permet de documenter mon évolution, de présenter mes projets et de démontrer ma progression.

Je suis ouvert à toute opportunité de stage, de mentorat ou de collaboration dans le domaine du numérique.

Mes Projets

Projet 1 : Citation biblique

Description : "Citation biblique" est mon premier projet solo, où j’ai créé un site web simple pour afficher des citations bibliques que j’apprécie. Ce projet m’a permis de me familiariser avec les bases du développement web, en utilisant principalement HTML et CSS pour la structure et le design. Le site présente une collection de citations qui sont régulièrement mises à jour.

Technologies utilisées : HTML, CSS

Objectifs : Ce projet a été l’occasion pour moi d’expérimenter avec la création d’une première page web et de comprendre les bases de la structuration d’un site.

Projet 2 : Permis

Description : "Permis" est un site web que j’ai développé pour aider les utilisateurs à se préparer à l’examen théorique du permis de conduire en Belgique. Le site présente une série de questions sous forme de quiz, permettant aux utilisateurs de tester leurs connaissances et de se préparer efficacement à l'examen. Ce projet m'a permis d'améliorer mes compétences en JavaScript et en gestion de formulaire.

Technologies utilisées : HTML, CSS, JavaScript

Objectifs : L’objectif principal de ce projet était de créer une interface interactive pour simuler l'examen théorique, tout en apprenant à manipuler les données de manière dynamique avec JavaScript.

Lien vers le projet : Voir le projet

Projet 3 : Serveur Web avec Raspberry Pi 4B

Description : Ce projet est en préparation. L'objectif est de configurer un serveur web sur un Raspberry Pi 4B. Une fois le serveur mis en place, il pourra héberger des applications web légères. Ce projet me permettra de me familiariser avec l’administration de serveur et l’hébergement de sites web sur un appareil compact.

Technologies utilisées : Raspberry Pi 4B, Raspberry Pi OS, Apache/Nginx, PHP (facultatif), MySQL/MariaDB (facultatif)

Objectifs : L’objectif principal de ce projet est de mettre en place un serveur web fonctionnel avec un Raspberry Pi 4B et d’acquérir des compétences pratiques en administration de serveur et en hébergement web.

Lien vers le projet : Ce projet n’a pas encore débuté.

Projets à venir

Projet 4 : Projets électroniques

Description : Je prévois de réaliser plusieurs projets électroniques en utilisant un Raspberry Pi 4B et un Arduino Uno R4 avec divers composants électroniques. Mon premier projet a été une page web permettant de contrôler une LED RGB grâce à Python et au Raspberry Pi, mais je compte aller plus loin en intégrant des capteurs, des moteurs, et d'autres modules électroniques. Je cherche également à collaborer avec d'autres personnes passionnées pour échanger des idées et apprendre ensemble.

Technologies envisagées : Raspberry Pi 4B, Arduino Uno R4, Python, Composants électroniques divers (LED, capteurs, moteurs, etc.)

Objectifs : Développer des projets interactifs et pratiques avec des composants électroniques, apprendre à intégrer des modules matériels avec des logiciels, collaborer avec d'autres passionnés pour partager des idées et des connaissances.

Estimation de début du projet : Ce projet commencera dès que j'aurai de nouveaux composants et de nouvelles idées pour expérimenter.
